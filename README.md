# Portfolio Website

## Overview

This is my personal portfolio website where I showcase my skills, projects, and provide a way for potential collaborators and recruiters to contact me. The site is designed to be clean, responsive, and user-friendly, and is built with HTML, CSS, and JavaScript. The website is deployed on Amazon S3 for fast and scalable hosting.

## Features

- A fully responsive design that adapts to various screen sizes.
- Sections for showcasing skills, projects, and a contact form.
- Projects include direct links to their respective GitHub repositories.
- Contact form that allows visitors to get in touch, with submissions sent to a Google Sheet using Google Apps Script.
- Deployed using Amazon S3 for reliable and fast hosting.

## Technologies Used

- **HTML**: Structure of the website.
- **CSS**: Styling for the layout, typography, and responsiveness.
- **JavaScript**: For handling dynamic content such as the contact form.
- **Amazon S3**: For hosting and deploying the website.
- **Google Apps Script**: Used to send form submissions to a Google Sheet.
- **Git**: Version control for managing changes to the project.
- **Font Awesome**: Icons used for visual appeal.
